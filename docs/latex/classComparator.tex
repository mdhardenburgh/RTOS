\hypertarget{classComparator_comparatorDescription}{}\doxysubsection{Comparator Description}\label{classComparator_comparatorDescription}
The T\+M4\+C123\+G\+H6\+PM microcontroller analog comparator compares two analog volatages and provides a binary output as the result. Note that not all comparators can drive an output pin, see Signal Description section below for more detials. The analog comparator can provide its output to a device pin, replacing the need for a discrete analog comparator on the board. The comparator can also flag the start of an A\+DC sample sequence. The analog comparator can also generate interrupts.

The T\+M4\+C123\+G\+H6\+PM microcontroller has two independent analog comparators with the following functions\+:
\begin{DoxyItemize}
\item Compare external pin input to external pin input or to internal programmable voltage reference
\item Compare a test voltage against any one of the following voltages\+:
\begin{DoxyItemize}
\item An individual external reference voltage
\item A shared single external reference voltage
\item A shared internal reference voltage
\end{DoxyItemize}
\end{DoxyItemize}

For more detailed information on the Analog \mbox{\hyperlink{classComparator}{Comparator}} please see page 1215 of the T\+M4\+C123\+G\+H6\+PM datasheet @ \href{https://www.ti.com/lit/ds/symlink/tm4c123gh6pm.pdf}{\texttt{ https\+://www.\+ti.\+com/lit/ds/symlink/tm4c123gh6pm.\+pdf}}\hypertarget{classComparator_comparatorSignalDescription}{}\doxysubsubsection{Comparator Signal Description}\label{classComparator_comparatorSignalDescription}
The following table lists the G\+P\+IO pin mux and physical pin number of the pins that can be muxed for use by the Analog \mbox{\hyperlink{classComparator}{Comparator}}. The {\ttfamily A\+F\+S\+EL} bit in the G\+P\+IO Alternate Function Select (G\+P\+I\+O\+A\+F\+S\+EL) register should be set to choose the Analog \mbox{\hyperlink{classComparator}{Comparator}} function. The number in parentheses is the encoding that must be programmed into the {\ttfamily P\+M\+Cn} field in the G\+P\+IO Port Control (G\+P\+I\+O\+P\+C\+TL) register to assign the Analog \mbox{\hyperlink{classComparator}{Comparator}} signal to the specified G\+P\+IO port pin. The positive and negative input signals are configured by clearing the D\+EN bit in the G\+P\+IO Digital Enable (G\+P\+I\+O\+D\+EN) register.
