\hypertarget{classQei_qeiDescription}{}\doxysubsection{Q\+E\+I Description}\label{classQei_qeiDescription}
The T\+M4\+C123\+G\+H6\+PM microcontroller has two Q\+EI modules. The Q\+EI module is used to calculate linar displacement, direction of rotation and speed. This is useful for robotics to provide control of motors. The Q\+EI has the following features\+:
\begin{DoxyItemize}
\item Position integrator that tracks the encoder position
\item Programmable noise filter on the inputs
\item Velocity capture using built-\/in timer
\item The input frequency of the Q\+EI inputs may be as high as 1/4 of the processor frequency (for example, 12.\+5 M\+Hz for a 50-\/M\+Hz system)
\item Interrupt generation on\+:
\begin{DoxyItemize}
\item Index pulse
\item Velocity-\/timer expiration
\item Direction change
\item Quadrature error detection
\end{DoxyItemize}
\end{DoxyItemize}
