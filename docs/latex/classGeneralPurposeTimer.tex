\hypertarget{classGeneralPurposeTimer_generalPurposeTimerDescription}{}\doxysubsection{General Purpose Timer Description}\label{classGeneralPurposeTimer_generalPurposeTimerDescription}
The T\+M4\+C123\+G\+H6\+PM microcontroller has 6 16/32-\/bit \char`\"{}short\char`\"{} general purpose timer blocks and 6 32/64-\/bit wide timer blocks. Each short timer block is split up into 2 16-\/bit timers (Timer A and Timer B). The A and B timer can be concatenated to make the short timer a single 32-\/bit timer. The same functionality goes for 32/64-\/bit wide timers as well. A\+DC can be triggered using a general purose timer. The General Purpose Timer module has the following capabilities\+:
\begin{DoxyItemize}
\item Twelve 16/32-\/bit Capture Compare P\+WM pins (C\+CP)
\item Twelve 32/64-\/bit Capture Compare P\+WM pins (C\+CP)
\item Daisy chaining of timer modules to allow a single timer to initiate multiple timing events
\item Timer synchronization allows selected timers to start counting on the same clock cycle
\item A\+DC event trigger
\item User-\/enabled stalling when the microcontroller asserts C\+PU Halt flag during debug (excluding R\+TC mode)
\item Ability to determine the elapsed time between the assertion of the timer interrupt and entry into the interrupt service routine
\end{DoxyItemize}

Efficient transfers using Micro Direct Memory Access Controller (Âµ\+D\+MA)
\begin{DoxyItemize}
\item Dedicated channel for each timer
\item Burst request generated on timer interrupt
\end{DoxyItemize}

Additionally, the short and wide timers have the following capabilities listed in the following table.

The prescaler is only available when the timers are used individually

A prescaler is an electronic counting circuit used to reduce a high frequency electrical signal to a lower frequency by integer division. The prescaler takes the basic timer clock frequency (which may be the C\+PU clock frequency or may be some higher or lower frequency) and divides it by some value before feeding it to the timer, according to how the prescaler register(s) are configured. The prescaler values that may be configured might be limited to a few fixed values (powers of 2), or they may be any integer value from 1 to 2$^\wedge$P, where P is the number of prescaler bits.

For more detailed information on the General Purpose Timer please see page 704 of the T\+M4\+C123\+G\+H6\+PM datasheet @ \href{https://www.ti.com/lit/ds/symlink/tm4c123gh6pm.pdf}{\texttt{ https\+://www.\+ti.\+com/lit/ds/symlink/tm4c123gh6pm.\+pdf}}\hypertarget{classGeneralPurposeTimer_generalPurposeTimerSignalDescription}{}\doxysubsubsection{General Purpose Timer Signal Description}\label{classGeneralPurposeTimer_generalPurposeTimerSignalDescription}
