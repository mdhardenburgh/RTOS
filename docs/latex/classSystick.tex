\mbox{\hyperlink{classSystick}{Systick}} is a system timer used for thread switching. \mbox{\hyperlink{classSystick}{Systick}} is genrally not used for general purpose timing. The \mbox{\hyperlink{classSystick}{Systick}} timer is a 24-\/bit clear-\/on-\/write, decrementing, wrap-\/on-\/zero counter.
