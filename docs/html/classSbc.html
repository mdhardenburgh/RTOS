Detailed Description

TM4C123GH6PM SBC Driver.

SBC Description

The SBC Provides system implementation information and system control, including configuration, control, and reporting of system exceptions.

For more detailed information on the SBC please see page 125 of the TM4C123GH6PM datasheet @ https://www.ti.com/lit/ds/symlink/tm4c123gh6pm.pdf

SBC Register Description

The SBC class contains a list of SBC registers listed as an offset relative to the hexadecimal base address of Core Peripherals 0xE000E000.

This Class lists and describes the System Control Block (SCB) registers, in numerical order by address offset. The SCB registers can only be accessed from privileged mode.

All registers must be accessed with aligned word accesses except for the FAULTSTAT and SYSPRI1-SYSPRI3 registers, which can be accessed with byte or aligned halfword or word accesses. The processor does not support unaligned accesses to system control block registers.
