DESCRIPTION

Class header for the system control unit for the Texas Instruments Tiva C ARM4F microcontroller, TM4C123GH6PM.

System Control Register Descriptions

All addresses given are relative to the System Control base address of 0x400F.E000. Registers provided for legacy software support only are listed in “System Control Legacy Register Descriptions” on page 424.

μDMA Register Descriptions

The register addresses given are relative to the μDMA base address of 0x400F.F000.
