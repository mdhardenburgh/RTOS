Detailed Description

TM4C123GH6PM QEI Driver.

QEI Description

The TM4C123GH6PM microcontroller has two QEI modules. The QEI module is used to calculate linar displacement, direction of rotation and speed. This is useful for robotics to provide control of motors. The QEI has the following features:

  • Position integrator that tracks the encoder position
  • Programmable noise filter on the inputs
  • Velocity capture using built-in timer
  • The input frequency of the QEI inputs may be as high as 1/4 of the processor frequency (for example, 12.5 MHz for a 50-MHz system)
  • Interrupt generation on:
    • Index pulse
    • Velocity-timer expiration
    • Direction change
    • Quadrature error detection

For more detailed information on the QEI please see page 1305 of the TM4C123GH6PM datasheet @ https://www.ti.com/lit/ds/symlink/tm4c123gh6pm.pdf
